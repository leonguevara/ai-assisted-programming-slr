## Original Search Specifications

Impact of AI-assisted code generation tools on software development, including GitHub Copilot, ChatGPT, Amazon CodeWhisperer, Claude Code, Gemini, and OpenAI Codex.

Research Questions:
RQ1: Impact on developer productivity
RQ2: Effect on code quality (correctness, maintainability, readability)
RQ3: Common evaluation metrics used in empirical studies
RQ4: Main benefits reported when using these tools
RQ5: Limitations, risks, and challenges (security, hallucinations, over-reliance)
RQ6: Research methodologies used (experiment, survey, case study)
RQ7: Trends and research gaps in the literature

Search Requirements:
- Empirical studies on AI-assisted programming and LLM-based code generation
- Papers evaluating tools like GitHub Copilot, ChatGPT, CodeWhisperer, or similar
- Studies with measurable outcomes (productivity, defect rate, time, accuracy)
- Published in peer-reviewed venues or reputable preprints (arXiv)
- Year range: 2021 to 2026
- English language only
- Retrieve maximum coverage, aiming for approximately 2000 papers
- Focus on sources: arXiv, IEEE, ACM, Springer, Elsevier, Scopus-indexed venues
- Include papers with accessible metadata (DOI or URL)
- Exclude opinion papers, essays, editorials without empirical data
- Exclude pure NLP studies not related to programming tasks
